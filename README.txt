Voici les étapes que vous devez suivre pour installer correctement l'application, la configurer et l'améliorer :

1. Création des dossiers côté serveur :
   - Créez un dossier principal.
   - Ajoutez les fichiers suivants dans ce dossier : coon.php, dashboard.php, get_departments.php, get_services.php, index.php, miseajour.php,
   presence.php, raports.php, search.php, stagiaire.php, stagiaires_competence.php, stagiaires.php, utilisateurs.php.

2. Création des dossiers suivants :
   - Créez les dossiers suivants dans le dossier principal : css, icon, pdf, photos.

3. Configuration des fichiers CSS :
   - Dans le dossier "css", ajoutez les fichiers suivants : index.css, style.css.

4. Configuration des fichiers d'icônes :
   - Dans le dossier "icon", ajoutez l'icône suivante : icon.ico.

5. Configuration des fichiers de photos :
   - Dans le dossier "photos", ajoutez les photos suivantes : snim.png, logout.svg.

6. Configuration des fichiers PDF :
   - Dans le dossier "pdf", ajoutez les fichiers suivants : competances_pdf.php, fpdf.css, fpdf.php, info.php, presence_pdf.php, Rapport_Final.php, snim.png.
   - Créez également un dossier "font" dans le dossier "pdf" et ajoutez les fichiers suivants : helvetice.php, helveticai.php, times.php, timesb.php.

7. Importation des tables dans la base de données "snim" :
   - Importez les tables suivantes dans la base de données "snim" :
     - Table "utilisateurs" : Cette table contient les utilisateurs avec leurs informations. Il y a un utilisateur administrateur avec les identifiants suivants :
       - Utilisateur : meyara
       - Mot de passe : 027134
     - Table "directions.sql"
     - Table "departements.sql"
     - Table "services.sql"
     - Table "les_stagiaires.sql"
     - Table "presences.sql"
     - Table "competances.sql"

Assurez-vous de suivre ces étapes avec précision pour installer correctement l'application, configurer les fichiers nécessaires et importer les tables dans la base de données.
